Accordion

A vertically stacked set of interactive headings that each reveal an associated section of content. Supports single and multiple open items.

Default
Grouped accordion with a bordered container. First item open by default.
Variant=Default
A design system built for speed and consistency, sourced directly from Figma variables and components.
Copy the component file from the site, drop it into your project, and import it wherever you need it.
Yes — all tokens are Figma variables. Update them in Figma and re-export to keep your code and design in sync.
Yes. Every component ships with light and dark token sets, toggled via a CSS class on the parent container.
tsx 
<Accordion type="single" defaultValue="item-1">
  <AccordionItem value="item-1">
    <AccordionTrigger>What is Espresso UI?</AccordionTrigger>
    <AccordionContent>
      A design system built for speed and consistency.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-2">
    <AccordionTrigger>How do I install it?</AccordionTrigger>
    <AccordionContent>
      Copy the component file and import it into your project.
    </AccordionContent>
  </AccordionItem>
</Accordion>

Alert Dialog

A modal confirmation dialog that interrupts the user with important content and expects a response. Supports default and destructive variants in two sizes, with full light and dark mode support.

Default
Neutral confirmation — horizontal layout with icon, title, body, and Cancel + Continue buttons.
Size=Default · State=Default

↓ Download
